# Zenarch

## Overview
Zenarch is a modern architecture portfolio web application designed to showcase architectural projects in a clean and visually engaging way. It allows visitors to explore featured works, while providing a structured platform for managing and presenting design portfolios professionally.

---

## How the Project Works
The platform displays architectural projects with detailed visuals and information, creating an interactive portfolio experience for users. An admin panel is integrated into the system, allowing authorized users to manage content efficiently. Admins can add, edit, and delete projects, as well as manage featured listings.

The system also supports multiple admin roles, enabling controlled access and better content management. The frontend ensures a responsive and smooth browsing experience, while the backend handles data operations, authentication, and role-based access control. MongoDB is used for flexible data storage, making it easy to scale and manage project data.
